# 어디서 찾았나요?

tailwind blog 템플릿을 이용해 블로그를 만들었는데 mdx 포맷을 기본적으로 사용중이었다.

# 왓 이스 디스

일단 공식 사이트를 확인해보니까 Markdown for the component era라고 적혀있다. 대충.. 마크다운 컴포넌트..? 일단 더 읽어보자.

더 읽어보니까 MDX는 markdown에 JSX를 같이 사용할 수 있는 맛있는 포맷이라고 한다. 또한 대화형 구성 요소를 포함할 수 있다고 한다.

예시 코드를 가져와보자.

```js
import {Chart} from './snowfall.js'
export const year = 2023

# Last year’s snowfall

In {year}, the snowfall was above average.
It was followed by a warm spring which caused
flood conditions in many of the nearby rivers.

<Chart year={year} color="#fcb32c" />
```

markdown과 jsx가 섞여있어서 되게 신기하다 ㅋㅋㅋㅋ
변수를 사용할 수도 있고 되게 편한 것 같다.
